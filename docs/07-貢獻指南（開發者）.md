# 07-貢獻指南（開發者）

## 開發環境

```bash
git clone https://github.com/billpwchan/futu_tick_downloader.git
cd futu_tick_downloader
python3 -m venv .venv
. .venv/bin/activate
pip install -U pip
pip install -e .[dev]
```

## 你送 PR 前要做的事

```bash
make lint
make test
```

## 開發原則

- 不破壞現有功能，預設行為維持相容。
- 任何行為變更必須同步文件、測試、runbook。
- 不提交 secrets；僅維護 `.env.example`。

## Commit / PR 建議

- 小步提交，訊息清楚。
- PR 描述請填：動機、變更、測試證據、風險。
- schema 或運維行為變更時，務必更新 docs 與 `CHANGELOG.md`。

完整社群規範請看：[`/CONTRIBUTING.md`](../CONTRIBUTING.md)
