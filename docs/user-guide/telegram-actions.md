# User Guide: Telegram 按鈕怎麼用

## 最短流程（收到 ALERT 時）

1. 先看主訊息「結論」與「影響」
2. 按 `🧾 近20分鐘日誌` 看是否持續
3. 按 `🗃 DB 狀態` 看 rows/max_ts/drift
4. 按 `🧯 建議/處置` 執行下一步

## HEALTH 訊息怎麼看

- 第一行：目前嚴重度（OK/WARN/ALERT）
- 第二行：結論（是否需要介入）
- 第三行：關鍵指標（市況、落庫、延遲、今日 rows、佇列）
- 第四行：資源指標（disk/load）

## 詳情模式

- `🔎 詳情` 可在同一則訊息展開/收合
- 展開後會看到較工程化欄位（吞吐拆分、symbol lag 等）
- 若內容過長，末尾會提示「內容已截斷，請用 🧾/🗃 看更多」

## 常見狀況

- 按鈕按了沒反應：請確認你在 `TG_ADMIN_USER_IDS` 中
- 提示逾時：表示日誌/DB 查詢超時，稍後重試
- 告警很多：可先按 `🔕 靜音 1h` 降噪（不影響 ALERT）
